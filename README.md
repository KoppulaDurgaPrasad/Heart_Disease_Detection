## 📌 Project Overview

Heart Disease Prediction System is a Machine Learning web application that predicts the likelihood of heart disease using patient health information.

The project follows a complete Machine Learning workflow:

- 🧹 Data Cleaning
- 📊 Exploratory Data Analysis (EDA)
- ⚙️ Feature Engineering
- 🤖 Model Training
- 📈 Model Evaluation
- 🌐 Deployment using Streamlit Cloud

## 🤖 Machine Learning Models Evaluated

| 🧠 Model | 🎯 Accuracy |
|-----------|------------|
| Logistic Regression | **86.41%** 🏆 |
| KNN | 85.33% |
| Naive Bayes | 85.33% |
| SVM (RBF Kernel) | 84.78% |
| Decision Tree | 79.35% |

### 🏆 Selected Model

**Logistic Regression** was selected because it achieved the highest accuracy among all evaluated models.

**Final Accuracy: 86.41%**
